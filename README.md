# Setup python virtual environment

Create virtual environment:
```bash
   python3 -m venv
```

Install packages from requirements.txt:
```bash
   pip install -r requirements.txt
```

Install flask:
```bash
   pip install flask
```

Install flask-WTF:
```bash
   pip install flask-WTF
```

Install flask-dotenv:
```bash
   pip install flask-dotenv
```
